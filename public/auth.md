## Primewayz UK Agent Authentication

Primewayz UK is a public marketing and digital support website for UK SMEs, founders, consultants, and service-led businesses.

## Current agent access

Primewayz UK currently exposes public website content, service pages, blog insights, sitemap, robots.txt, and llms.txt for discovery.

No OAuth-protected public API is currently offered for third-party agents.

## Agent usage guidance

Agents may read public pages to help users understand Primewayz UK services, blog insights, website audit options, and contact routes.

Agents should not submit forms, create leads, or make business commitments on behalf of users without explicit user confirmation.
